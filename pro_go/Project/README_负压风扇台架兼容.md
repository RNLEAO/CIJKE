# 负压风扇台架兼容说明

## 主要变化

1. 基于队长的四路电感版本 `53e7947` 增加负压风扇控制模块，完整保留原有屏幕、电感、IMU、编码器和电机安全设置。
2. 使用 `P33 → Q2 → 负压风扇` 驱动路径，PWM频率固定为17kHz，占空比固定为30%，单次输出约1秒。
3. 新增第17页 `NP BENCH` 台架页面，支持 `EN`、`ARM`、`FIRE`、`RESET` 操作，并使用 `WAIT`、`READY` 显示状态。
4. 增加停车状态保护、单次触发锁定和30秒冷却，避免误触发和连续启动。

## 台架操作

使用P71切换到第17页 `NP BENCH`，P73选择项目，P72执行操作：

1. 选择 `EN` 并按P72，使模块进入使能状态。
2. 选择 `ARM` 并按P72，完成布防。
3. 选择 `FIRE` 并按P72，风扇以固定参数运行约1秒。
4. 页面显示 `WAIT` 时等待冷却结束。
5. 选择 `RESET` 并按P72；复位成功后显示 `READY`。
6. 再次执行 `ARM → FIRE`，开始下一次测试。

一次触发结束后 `ARM` 自动变为 `OFF`，属于防止重复启动的安全行为。

## 安全逻辑

P33实际输出必须同时满足：

- 模块已经使能
- 模块已经布防
- 收到一次有效的 `FIRE` 请求
- 行走状态 `pwm_state == 0`
- 未处于故障、冷却或锁定状态
- 状态机处于 `PREPARE` 或 `HOLD`

任意条件失效时，P33占空比立即归零。
