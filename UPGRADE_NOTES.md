# V3.2.4 四路电感升级说明

分支：`feature/v3.2-four-inductance-upgrade`

日期：2026-07-12

## 本次三项主要变化

### 1. 逐飞库从旧版 V1.0 全面升级到 V3.2.4

- 当前库版本已通过 `pro_go/Libraries/doc/version.txt` 核实为 `V3.2.4`。
- 本次升级覆盖 `zf_common`、`zf_device`、`zf_driver`、`zf_components` 等库模块。
- 工程芯片目标为 STC32G12K128，并使用 Keil C251 工具链。
- 源码中出现的 `@since v1.0` 是个别 API 的历史注释，不代表当前整套库版本。

这是本次改动范围最大、底层影响最广的一项变化。

### 2. 五路电感改为四路电感

- `L` 左外：P00。
- `LM` 左内：P01。
- `RM` 右内：P05。
- `R` 右外：P06。
- 不再依赖 P10 中间电感。
- 新增四路独立采样、滤波、归一化、人工标定、Flash 配置版本管理和串口诊断。
- 直角识别按实车十组采样数据重新设置阈值，并保留连续周期确认。
- 环岛状态机保留，但环岛阈值尚待真实环岛样本进一步验证。

人工采样和最终参数记录见：

`pro_go/四路电感人工标定参数与实测记录.docx`

### 3. 屏幕和按键交互优化

- 调整页面布局、字号和行间距，减少文字拥挤与遮挡。
- STOP 状态移动到右下角，避免遮挡主要数据。
- 优化页面切换和按键事件处理，区分短按、长按与组合操作。
- 无屏幕时仍可通过无线串口完成状态检查和前期调试。
